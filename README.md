# Sleep Health and Lifestyle Analysis

## Overview
This project analyzes the relationship between sleep quality and lifestyle factors such as stress, physical activity, BMI, occupation, and sleep disorders using Python and data visualization techniques.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy

## Analysis Performed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Correlation Heatmap
- Stress vs Sleep Quality Analysis
- Physical Activity vs Sleep Quality Analysis
- Sleep Quality by BMI Category
- Sleep Disorder by Occupation
- ANOVA Statistical Test

## Key Findings
- Higher stress is associated with lower sleep quality.
- Physical activity positively impacts sleep quality.
- BMI influences sleep quality.
- Sleep disorders vary across occupations.

## Conclusion
A healthy lifestyle characterized by low stress, regular physical activity, and a healthy BMI contributes to better sleep quality.
